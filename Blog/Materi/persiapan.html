Persiapan Belajar HTML


Pada artikel sebelumnya kita telah berkenalan dengan apa itu HTML dan fungsinya. Sekarang kita akan langsung mempelajarinya. Tapi untuk mempelajari ini kita perlu beberapa persiapan terlebih dahulu.


Hal yang perlu dipersiapkan untuk mempelajari ini tidaklah banyak hanya perlu editor kode dan browser.


Untuk kalian yang menulis kodenya melalui laptop atau pc saya sarankan mendownload vs code dan bagi kalian yang tidak memiliki laptop atau komputer kalian bisa download acode di play store atau editor kode lain jika kalian pengguna ios. saya kurang tahu editor kode yang ada di app store.


atau kalian juga bisa menggunakan editor kode online seperti code pen atau OneCompiler. Tapi saran saya lebih baik kalian download kode editor sendiri.


Cara download VS Code



  1. Pergi ke browser

    2. Kalian dapat menggunakan browser apapun, disini saya menggunakan google chrome.

    Setelah kalian membuka browser ketikan " Download visual studio code " atau kalian bisa klik link ini download vs code.



  2. Download vs code

    3. Setelah sampai di halaman download. Download vs code sesuai operating system yang kalian gunakan. Lalu tunggu hingga proses selesai.
