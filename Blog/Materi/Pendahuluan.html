Pendahuluan


Halo semuanya selamat datang di belajar HTML autodidak. Mari kita mulai dengan kenapa belajar html? karena html adalah pondasi utama dari sebuah website. Selain itu juga, menurut saya html adalah langkah yang baik bagi pemula yang ingin belajar pemrograman.


Berdasarkan dari pengalaman saya dulu saya memulai dengan mempelajari bahasa C dan saya bingung dengan apa maksud dari semua ini. maksud saya tipe data, variabel, dan masih banyak lagi.


Setelah itu saya pindah mempelajari bahasa python dan sama saya bingung saya ingin membuat sebuah vidio game tapi yang keluar dari program saya hanyalah serangkaian huruf pada terminal. Sampai akhirnya saya bertemu tutorial web programming dan kebetulan sedang menjelaskan tentang html dan css.


Setelah mempelajarinya saya tau apa yang dilakukan kode saya dan apa yang bisa saya buat dengan ini.


Mungkin bagi kalian yang bilang " ah nggak kok python juga bisa dibikinin apa aja " memang iya tapi bagi seorang yang benar-benar pemula akan bingung dan cepat bosan, karena memiliki roadmap yang panjang.


Apa itu HTML?

HTML atau hayper text markup language adalah bahasa markah yang mengatur bagaimana konten ditampilkan di sebuah website. Ya saya minta maaf HTML bukanlah bahasa pemrograman.


Tapi jangan khawatir penulisan kode HTML sama seperti menulis bahasa pemrograman pada umumnya yaitu memerlukan ketelitian dan fokus. Ditambah tidak ada error pada html jadi dibutuhkan ketelitian untuk mencari masalah pada kode seperti lupa menulis tutup kurung siku pada sebuah tag.


Dan juga kenapa di awal tadi saya mengatakan bahwa HTML adalah bahasa yang cocok dipelajari oleh pemula. Karena selain mudah ini memiliki urutan yang jelas. Maksud saya setelah mempelajari html kamu akan mempelajari css untuk membuat tampilan web menjadi menarik dan setelah itu javascript ini adalah bahasa pemrograman yang akan membuat website kalian menjadi interaktif.


Dengan menguasai ketiga hal ini mungkin kamu bisa disebut front end developer. Oiya mungkin saat kalain selesai mempelajari javascript dan mencoba mempelajari bahasa lain kalian tidak akan terlalu kesulitan mempelajarinya, karena kalian sudah memiliki pengetahuan dasar tentang pemrograman.


Sudah tidak sabar untuk memulai ? tutorial HTML ini saya bagi menjadi empat bagian. yang membahas tentang dasar html, menambahkan multimedia, membuat table, dan isi tag head.
